# カーソル追従型スムーズスクロール（説明用メモ）

実装は **Campaign の盤** に統合済みです。単体デモシーンは置かず、次を実行すると挙動を確認できます。

- **シーン**: `res://assets/scene/test1/hex_field_stage.tscn`（メニュー → New Game → **Campaign** でも入れる）

## コードの対応関係

| 説明の言葉 | 実装 |
|------------|------|
| カーソル（マス単位） | `srpg_map_cursor.gd`（`MapCursor` ノード） |
| 視点（端で追う・なめらか） | `srpg_camera_dead_zone.gd`（`Camera2D`） |
| 六角マップの数式 | `hex_grid.gd`（`HexFieldGrid`） |
| Esc / 画面下ヒント | `hex_field_stage.gd`（ルートのみ） |

## 実行確認

1. プロジェクトを開く  
2. **Campaign** で盤に入る、または `hex_field_stage.tscn` を開いて **F6**  
3. 矢印でカーソル移動 → 画面中央付近ではカメラはほぼ止まり、**端に寄るとスーッと追従**する

## 口頭説明用の骨子（イメージ重視）

- **カーソル**＝「どのマスを見ているか」の印（マウスカーソルに近い）  
- **カメラ**＝「世界のどこを画面に映すか」  
- **中央に常に固定しない**＝画面の真ん中にカーソルを張り付け続けない  
- **デッドゾーン**＝「この内側ではカメラは動かない」箱。外に出そうになったら目標だけ動かす  
- **スムーズ**＝毎フレーム `lerp` で寄せていく（パッと切り替えない）

詳しい比喩はチーム用ドキュメントに任せる場合、このファイルは **「どのスクリプトがそれに当たるか」** の索引として使える。
