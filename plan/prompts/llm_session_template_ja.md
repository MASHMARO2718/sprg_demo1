# LLM 向けセッション・プロンプト（日本語・変動部用）

## 固定ルールとプロジェクト事実

**決定事項・事実の全文は [llm_session_bilingual.md](llm_session_bilingual.md) に集約しています。**  
新しいチャットや方針変更時は、まずそのファイルの **「貼り付け用・固定部」**（日本語＋フランス語ブロック）をコピーして送ってください。

---

## 毎回ここだけ書く（日本語）

固定部を既に送っている場合、または変動だけ更新する場合に使います。

```
【このセッションの言語】
- 指示: <日本語 など>
- 返答: <日本語 など>

【現在の進捗】
- 完了: <箇条書き>
- 進行中・次: <箇条書き>
- ブロッカー: <あれば / なし>

【今回の依頼】
<具体的なタスク。成果物・期待する挙動>

【参照パス（任意）】
<例: plan/plan001/ja/architecture_decisions_ja.md の該当節>
```

---

## 超短縮（同一チャットの続き）

```
【差分】<前回から変わった進捗のみ>
【依頼】<今回のタスク>
```

方針転換や長い中断のあとは、`llm_session_bilingual.md` の固定部を再度貼ると安全です。
